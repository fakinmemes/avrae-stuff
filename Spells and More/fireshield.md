!fireshield to cast the spell. It adds an attack and gives resistance based on the type of shield chosen. You can add the following arguments below. They can be added together.

!fireshield [cold/warm] casts the shield with that specific type of shield. If no argument is given no effect will be added.

!fireshield [-l #] will cast it with the spell slot level given. Use when you have no more 4th-level spell slots to prevent an error from returning, or if you don't want to spend a 4th-level spell slot for it.

!fireshield [-i] will cast it without consuming a spell slot.
